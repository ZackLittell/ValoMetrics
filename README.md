# ValoMetrics

ValoMetrics is a web-based analytics platform that will analyze Valorant esports performance. The system will collect match data and will analyze and display it in a way that will allow users to evaluate players, teams, and map data.

## Features

Player Performance Dashboard
Team Comparison Tool
Agent META Analysis
Map Performance Analysis
Custom Ranking System
